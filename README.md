# 🚀 AI Resume Analyzer

> **Intelligent Resume Analysis for Smarter Hiring Decisions**

AI Resume Analyzer is a modern AI-powered web platform that helps students, recruiters, and companies analyze resumes using **NLP, ATS scoring, semantic matching, and real-time AI insights**.

Upload a resume → Extract skills → Compare with Job Description → Generate ATS Score → Get AI Suggestions 📄✨

---

## ✨ Features

- 📄 Resume Upload (PDF/DOCX)
- 🧠 AI Resume Analysis
- 🎯 ATS Score Generation
- 🔍 Job Description Matching
- 📊 Recruiter Analytics Dashboard
- 🤖 AI Resume Feedback
- 🏆 Candidate Ranking System
- 💬 AI Chatbot Assistant
- 🔐 Secure Authentication
- 🌙 Modern Dark/Light UI

## 🚀 Getting Started

```bash
git clone https://github.com/your-username/ai-resume-analyzer.git
cd ai-resume-analyzer
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run dev
```
